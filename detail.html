Software Reverse Engineering

Software Service Reverse Enginnering
Client Navico
Date January 2020
Tools Olly Dbg & ASM .
Description In the last two years, Navico migrate all DB Servers, from SQL Server 2015 to SQL Server 2019.
But they was obligated to mantain one Server with and old database, because they have two old applications, with connections to this old database.
The aplicattions was developed in visual basic 6.0, 10 years ago with a DB connection string "harcoded" in source. Over the years they lost the source code and had no chance of changing the connection to the database.
I analyse the exe files with Ollydbg to add a new db connection string and re-assemble the exe file.
